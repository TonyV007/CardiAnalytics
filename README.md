# ❤️ CardiAnalytics

## 📊 Overview

CardiAnalytics is a data-driven healthcare analytics project focused on analyzing cardiovascular data using a complete pipeline approach.

## ⚙️ Project Structure

* `01_Python_Pipeline/` → Data preprocessing & ETL
* `02_Database/` → Database design & storage
* `03_SpringBoot_API/` → Backend API services
* `04_PowerBI/` → Visualization dashboards
* `05_Dataset/` → Raw & processed datasets
* `06_Reports/` → Insights & documentation
* `07_Models/` → Machine learning models

## 🚀 Tech Stack

* Python
* SQL
* Spring Boot
* Power BI
* Machine Learning

## 🎯 Objective

To build an end-to-end healthcare analytics system for better insights into cardiovascular conditions.
